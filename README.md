
# Bandzzz Beauty Go — Git Repo (Vercel Import)

## Local Dev
```bash
npm install
npm run dev
```

## Deploy (Vercel → Import Git)
1) Push this repo to GitHub.
2) Go to https://vercel.com/new → Import Git → select the repo.
3) Build command: `npm run build` (auto)
4) Output: `dist` (auto)
5) (Optional) Env vars: STRIPE_SECRET_KEY, BUTTONDOWN_API_KEY
6) Deploy → get live URL.
